本專案是一個 Jupyter Notebook 檔案，旨在實作與比較傳統自然語言處理（NLP）方法與現代 AI 模型在文本處理任務上的表現。

## 如何執行

### 1\. 環境設定

本專案基於 Python 環境，建議使用 Google Colab 開啟 `.ipynb` 檔案以獲得最佳體驗。

### 2\. 安裝依賴套件

執行 Notebook 中的第一個儲存格，以安裝所有必要的 Python 套件：

```bash
pip install jieba scikit-learn openai
```

### 3\. 設定 API 金鑰

在執行 Part B（現代 AI 方法）之前，請務必在指定的儲存格中填入您自己的 OpenAI API 金鑰。

### 4\. 執行程式碼

依照 Notebook 儲存格的順序，從上到下依次執行即可查看各個任務的實作成果與比較。
